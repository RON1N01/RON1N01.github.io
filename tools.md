### Entra Log Zulu Converter (ELZC)

Cross-platform CLI utility for converting Entra ID sign-in and audit logs from UTC (Zulu) to PST or PDT for accurate investigation timelines.

**Features:**
- Linux (x86_64) and macOS (Apple Silicon) standalone binaries  
- Flag-based CLI operation (`-s` for PST, `-d` for PDT)  
- No Python required for end users  

**GitHub:**  
https://github.com/RON1N01/entra-zulu-converter
